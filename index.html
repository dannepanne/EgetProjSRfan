Sega Rally Championship is a 1994 racing video game developed by Sega AM3 and published by Sega. Originally released for arcades using the Sega Model 2 board, it was converted to the Sega Saturn in 1995 and Windows in 1997. The unique selling point of Sega Rally Championship was the ability to drive on different surfaces, with different friction properties, with the car's handling changing accordingly. As the first racing game to incorporate this feature, Sega Rally Championship is considered to be one of the milestones in the evolution of the racing game genre. It was also an early rally racing game and featured cooperative gameplay alongside the usual competitive multiplayer.

Arcade

The arcade version was released in October 1995 and was directed by Kenji Sasaki, a former Namco employee known for his work on Ridge Racer.

Saturn

The Saturn version was released in December 1995. It was the first home console port of the game and just a couple of months later got an enhanced release with support for the 3D Control Pad.

Windows 95

The Windows 95 version was released in January 1997, with a 30fps cap and no controller support. There is also a rare version with Direct3D support.

Playstation 2

The Playstation 2 version was released in January 2006 as a pack in with Sega Rally 2006. It was only released in Japan and is the only port of the game that runs at 60fps.

Lancia Delta Group A

The Lancia has a good top speed and acceleration but worse handling than the Toyota

Desert

The first course in the championship, easy with few sharp turns. Set in a savannah environment with animals along the track side.

Forest

The second course in the championship, now it gets harder with more and sharper turns. Set in a northern hemisphere forest.

Mountain

The final and hardest course in the championship, a (un)healthy dose of hairpin turns. Obviously set in a mountainous region.
